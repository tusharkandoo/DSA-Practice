* 💡 Interview Preparation
* 🏆 Competitive Programming
* 📚 Academic Learning
* 🚀 Problem Solving Skills

Clone the repository:

```bash

```

Open the folder:

```bash

```

Compile and run any C++ file:

```bash
g++ filename.cpp
./a.out
```
